# SensoryFlow (感官流) - Chrome 应用商店上架信息

## 1. 基本信息 (Basic Info)
- **扩展名称**：SensoryFlow (感官流)
- **简短说明**：为您的网页注入灵魂。根据场景适配视觉滤镜与动态特效，重塑数字阅读与工作感官。
- **详细描述**：
  SensoryFlow 是一款打破传统浏览体验的视觉增强工具。它将网页从死板的像素堆砌转化为一场沉浸式的感官之旅。

  ### 核心功能：
  - **极客模式**：为 GitHub 和技术站点注入赛博朋克能量。
  - **灵感模式**：将设计门户化作清澈的水池，伴有游鱼与涟漪。
  - **悦读模式**：让长文阅读回归纸质质感，守护视力，专注内容。
  - **禅意模式**：落花与宁静，在嘈杂的互联网中寻找一片净土。
  - **银河模式**：在繁星与流星间穿梭，感受宇宙的宏大。

  ### 进阶控制：
  - **专注聚光灯**：通过沉浸式光晕引导您的阅读注意力。
  - **滚动速度线**：为网页滚动赋予电影般的动感。
  - **智能性能保护**：内置 FPS 监测，确保在任何设备上都能流畅运行。

  ### 隐私承诺：
  SensoryFlow 不收集任何个人数据。所有处理均在本地浏览器中完成，安全、私密、快速。

## 2. 类别与关键词 (Category & Keywords)
- **类别**：生产力 / 无障碍

- **scripting / activeTab**：用于在您访问的页面上注入视觉层。
